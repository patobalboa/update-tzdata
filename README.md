# Update tzdate

This Script is for update tzdata from archives of iana.org in Centos

## How to use.

Clone this repository

```
git clone https://github.com/patobalboa/update-tzdata.git

```
Run the following commands
```
cd update-tzdata
chmod +x update-tzdata.sh
./update-tzdata.sh
```
